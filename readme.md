# Instrucciones de ejecución:
1. Instalar el gestor de dependencias Poetry:
```
pip install poetry
```
2. Ubicarse en la carpeta /backend y ejecutar:
```
poetry install
``` 
y posteriormente:
```
uvicorn src.api.app:app --reload
```
